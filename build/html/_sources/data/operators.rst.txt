=====================
Operators
=====================

Ruby has a built-in modern set of operators. Operators are a symbol which is used to perform different operations. **For example, +, -, /, *, etc.**

1. Types of operators:
======================

   - Unary operator
   - Airthmetic operator
   - Bitwise operator
   - Logical operator
   - Ternary operator
   - Assignment operator
   - Comparison operator
   - Range operator

2. Unary Operator
=================

Unary operators expect a single operand to run on.
